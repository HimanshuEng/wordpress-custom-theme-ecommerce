# WordPress Custom Theme ‚Äì E-commerce Homepage

This project is a custom WordPress theme developed as part of a Web Developer assignment.

## Features

- Fully mobile responsive design (based on provided mockup)
- Dynamic Announcement Bar
- Dynamic Hero Section
- Dynamic Product Grid (with image, price, and CTA button)
- Managed via WordPress Admin Panel
- User-friendly CMS structure for non-technical users

## Technologies Used

- WordPress
- PHP
- HTML5
- CSS3
- Bootstrap (if used)
- Custom Post Types (for Products)

## CMS Structure

- Announcement content editable from admin
- Hero section content editable from admin
- Products managed through WordPress dashboard
- No hardcoded frontend content

## Setup Instructions

1. Install WordPress
2. Copy theme folder into `wp-content/themes`
3. Activate theme from WordPress Admin
4. Configure content from dashboard
